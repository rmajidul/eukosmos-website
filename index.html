// Why EuKosmos

Built by a researcher,
for researchers

Most reference managers treat your data as a service. EuKosmos treats it as yours. Everything saves to a plain folder — JSON files you can read, copy, back up with git, and access without EuKosmos if you ever need to.

✓
DOI-first enrichment cascade

Paste a DOI and EuKosmos queries all 6 databases in sequence, filling every field it can find — authors with ORCIDs, abstract, BibTeX, citations, OA links.

✓
SQLite + JSON hybrid

Data lives in a reliable SQLite database for performance, with human-readable JSON exports always kept in sync alongside it.

✓
Browser extension

Add papers and save PDFs from any publisher page in one click — using your institutional access, no CAPTCHA, no manual renaming.

// DOI Enrichment Cascade

Every major database.
One paste.

Paste a DOI (or arXiv ID, ADS bibcode, InspireHEP key) and EuKosmos queries all databases in order, filling the richest metadata possible. No duplicates — each database contributes only what the others don't have.

ADS / SciX → InspireHEP → CrossRef → OpenAlex → DataCite → Semantic Scholar
ADS / SciX
NASA Astrophysics Data System. Canonical source for astronomy & astrophysics. Provides bibcodes, verified BibTeX, readCount, arxivClass, doctype.
Astronomy Astrophysics
InspireHEP
The primary database for high-energy physics. Provides texkeys, InspireHEP BibTeX, SCOAP3 OA PDFs, collaboration info.
HEP Particle Physics
CrossRef
DOI registration agency. Provides ISSN, publisher, license, funder information, author details with ORCID identifiers.
All fields
OpenAlex
Open scholarly graph. Provides OA status & URL, citation count, concepts, subject classification, PMC links.
All fields Open Access
DataCite
Metadata for research datasets and software. Provides resource type, funding info for non-journal outputs.
Datasets Software
Semantic Scholar
AI-powered research graph. Provides TLDRs, influential citation counts, field-of-study classification, S2 IDs.
All fields AI / CS
arXiv
Open access preprint server. Direct preprint PDF download, category classification, version history.
Physics CS Math
PubMed / PMC
NLM biomedical literature. PMID and PMCID identifiers, open access full text for NIH-funded research.
Biomedical Life Sciences

Fetch log shows exactly which database contributed each field — full transparency, always.

// How It Works

Up in three minutes.

EuKosmos runs entirely on your machine. No account, no cloud, no subscription. Just Python and a browser.

01
Download

Download EuKosmos for Mac or Windows. Or clone from GitHub if you prefer.

02
Run

Open Terminal. Run python3 serve.py. EuKosmos starts on localhost:8080.

03
Add papers

Paste a DOI or arXiv ID. EuKosmos fetches everything automatically from all databases.

04
Research

Write notes, track ideas, manage your CV, plan collaborations. Everything in one place.

QUICK START
# Clone and run
git clone https://github.com/rmajidul/eukosmos
cd eukosmos
python3 serve.py
# Open Chrome → http://localhost:8080
// Browser Extension

One click.
Paper saved.

The EuKosmos Chrome extension detects papers on any publisher page, checks if it's in your library, and saves the PDF using your browser's authenticated session — institutional access, VPN, whatever you have. No CAPTCHA. No manual renaming. Files saved as Author_Bibcode_Original.pdf automatically.

● Publisher pages detected automatically — Springer, Elsevier, Nature, APS, IOP, ACS, arXiv, InspireHEP, ADS and more
● New papers added with full enrichment — DOI cascade runs automatically on add
● Real-time sync — EuKosmos updates instantly without page refresh via Server-Sent Events

* Zotero's browser extension is mature with 300+ site-specific translators built up over many years. EuKosmos's extension is newer and expanding publisher coverage with each release.
** EuKosmos extension detects papers on any publisher page and saves PDFs using your browser's authenticated session — including institutional access and VPN. Coverage is actively expanding.
† EuKosmos does not have a built-in PDF reader — by design. PDFs open in your operating system's default viewer (Preview, Acrobat, Skim, Okular — whatever you prefer and already know). Annotations you add there are saved to the file. Next time you open the PDF from EuKosmos, your annotations are exactly where you left them. We think your favourite PDF tool is better than anything we'd build in.
